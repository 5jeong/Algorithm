# Hash 알고리즘

### 해시 선언  
HashMap<Character,Integer> map = new HashMap<>(); 
### 해시 Method   
* 해시에 값넣기   
→ map.put('a',3);     
  
* 해시의 key값   
map.keySet();   
  
* a(key)값의 value값   
map.get('a');   
  
* key의 value값의 default값을 0으로 설정  
map.getOrDefault(x,0)  
  
* key값이 포함되어있는지 확인(True,False)   
map.containsKey('a');   
  
* key의 개수   
map.size();   
  
* key값 삭제   
map.remove('a');
